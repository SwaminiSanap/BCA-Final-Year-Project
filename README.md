📦 Transport Website – Final Year Project
📌 Project Overview

This project is a Transport Service Website developed as my Final Year Project.
The website provides complete information about transport services with a simple, user-friendly interface and backend support for future scalability. It is designed to help users easily understand services and directly contact the transport provider.

🎯 Objectives

To design a responsive and informative transport website

To provide service details clearly to users

To enable direct contact and call options

To implement backend connectivity using Python Flask and MySQL

🌐 Website Pages

Home Page (index.html)
Displays an overview of the transport services and business details.

Services Page (services.html)
Lists all available transport services with clear descriptions.

Contact Page (contact.html)
Provides direct call options and contact details for quick communication.

About Page (about.html)
Contains information about the transport company, owner details, and working structure.
